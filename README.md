# HPO Explorer

**Live app:** **[https://hpoexplorer.com](https://hpoexplorer.com)**

> **Not the R package.** This project is **not**
> [neurogenomics/HPOExplorer](https://github.com/neurogenomics/HPOExplorer) (R/Bioconductor).
> HPO Explorer is a **free web application** for browsing the Human Phenotype Ontology and
> extracting HPO terms from clinical notes — entirely in your browser, with no install and no
> server upload of note text.

## What it does

- Interactive HPO browser (search, hierarchy, graph) for ~19,800 phenotype terms
- **Clinical note → HPO** extraction with negation, family history, and uncertainty handling
- Phenotypic similarity, candidate-disease ranking, and gene prioritization (client-side)
- Export to CSV, JSON, and GA4GH Phenopacket
