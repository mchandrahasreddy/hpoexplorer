# HPO Explorer (hpoexplorer.com)

**Free web app for the [Human Phenotype Ontology](https://hpo.jax.org) — browse ~19,800 HPO terms and extract phenotypes from clinical notes in your browser.**

**Live app:** **[https://hpoexplorer.com](https://hpoexplorer.com)**

> **Not the R package.** This repository is **not**
> [neurogenomics/HPOExplorer](https://github.com/neurogenomics/HPOExplorer) (R/Bioconductor).
> **HPO Explorer** is a standalone **web application** at **hpoexplorer.com** for deep phenotyping,
> clinical-note-to-HPO extraction, and phenotype-driven disease/gene analysis — with no install
> and no server upload of note text.

**Keywords:** Human Phenotype Ontology, HPO, clinical NLP, phenotype extraction, clinical note to HPO,
deep phenotyping, rare disease, genomic diagnosis, bioinformatics, Phenopacket, privacy-preserving NLP.

## What it does

- Interactive HPO browser (search, hierarchy, graph) for ~19,800 phenotype terms
- **Clinical note → HPO** extraction with negation, family history, and uncertainty handling
- Optional in-browser semantic matching for lay phrasing
- Phenotypic similarity, candidate-disease ranking, and gene prioritization (client-side)
- Export to CSV, JSON, and GA4GH Phenopacket
